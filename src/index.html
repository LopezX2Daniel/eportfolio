Unit 6

Development Team Project: Project Report

In this collaborative project, my colleagues and I designed a complete database architecture for a retail client called EPL Fan Fever Ltd. The goal was to provide a scalable solution integrating e-commerce and physical stores. We developed a cloud infrastructure using AWS EC2, PostgreSQL, and Flask/Python, with security layers. We designed as well a robust ETL pipeline using Apache Airflow for data validation, transformation, and loading, implementing deduplication techniques through hashing, anomaly detection with z-scores, and referential integrity validation.

We managed the entire project using Kanban methodology to systematically track progress and workflow. Through the Trello platform, we organised tasks, assigned responsibilities, and scheduled regular meetings to discuss outstanding issues and align on project deliverables. This collaborative approach proved instrumental in maintaining team coordination and successfully achieving the final project version.
